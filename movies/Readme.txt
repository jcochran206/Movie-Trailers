Project Title: Movie Trailer Project

You will write server-side code to store a list of your favorite movies, including box art imagery and a movie trailer URL. You will then use your code to generate a static web page allowing visitors to browse their movies and watch the trailers.


Getting Started

1) download python 2.7 and go thru installation process 
2) ensure all files are in movies folder 
3) open python idle 
4) open file entertainment_center.py
5) run entertainment_center.py
6) wait for web browser to open 
7) view web page and visually verify movie posters and items are available 
8) click on movie poster and enjoy trailer
